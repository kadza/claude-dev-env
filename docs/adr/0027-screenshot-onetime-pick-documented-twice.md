# One-time destination pick documented in the README and echoed by setup.sh

The one-time `Other Location…` pick is documented in a README subsection *and* echoed as a next-step reminder at the end of `setup.sh` (matching the existing OAuth-token note). It's the single manual, non-scriptable step in the whole flow ([0020](./0020-screenshots-non-default-save-to-destination.md)); if it isn't surfaced, screenshots silently won't appear. Rejected: README only (easy to miss); `setup.sh` echo only (no lasting reference once the terminal scrolls away).
