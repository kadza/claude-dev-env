description: Read the newest screenshot from the shared inbox
---

Read the newest file by mtime at the top level of `~/.claude-shots/` and view it. After you've used it, `rm` that file — the folder is a transient inbox, so keep it clean. If the folder is empty, say so and ask the user to save a screenshot there (on macOS: Cmd+Shift+5 → Save to → the `claude-shots` destination) rather than guessing.
